# My Hyprland Dotfiles

This repository contains my personal **Hyprland setup** and related configuration files, including Waybar, Wofi, Swaync, wlogout, and custom scripts.  
It is based on [Elifouts/Dotfiles](https://github.com/elifouts/Dotfiles) (GPL licensed) and includes my own modifications for themes, keybindings, and workflow enhancements.


## Features
- **Hyprland**
  - Custom keybindings and autostart scripts
  - Performance tweaks (blur adjustments)
  - Workspace and window management enhancements
- **Waybar**
  - Global taskbar showing windows from all workspaces
  - Music player integration
  - Bluetooth and battery icon customizations
- **Wofi**
  - Custom launcher configuration
- **Swaync**
  - Charger and battery notifications
- **wlogout**
  - Custom power menu
- **Pywal**
  - Dynamic color schemes and wallpaper scripts
- **Custom scripts**
  - Various scripts to enhance workflow and notifications

## Dependencies
- Hyprland
- Waybar
- Wofi
- Swaync
- wlogout
- Pywal (optional but recommended)

> Note: These configurations are tailored to my system and may require adjustments for other machines.
